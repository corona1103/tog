## 一、需求背景

**已确认的问题：**

1. 抢答、随机点名、小组讨论、智能体对话、互动课件 等互动可能无数据留存
2. 现有学情仅一个聚合卡片（次数/参与率/正确率），无法按类型细分
3. 各功能数据独立，无法形成学生单课完整画像
4. 管理者缺教师维度的课堂工具使用数据

---

## 二、需求目标

| 目标 | 使用者 | 用户场景 |
|:-----|:-------|:---------|
| 所有互动类型100%产生数据记录 | 系统自动 | 课中 |
| 对课堂行为分类统计，独立指标和展示 | 系统自动 | 课后聚合 |
| 班级间关键指标可排序对比 | 校长/教务 | 周会/学期末 |
| 教师按周月查看班级或个人的课堂表现趋势 | 教师 | 下课后/教研会 |

## 三、课堂行为分类体系

### 3.1 分类总览

| 类别 | 教学目标 | 参与范围 | 包含行为 | 核心指标 |
|:-----|:---------|:---------|:---------|:---------|
| **知识检测** | 检验知识点掌握 | 全班 | 随堂测、互动题（客观+主观）、单词小测 | 正确率、知识点掌握度、答题耗时 |
| **开放检验** | 检验课堂吸收（无预设知识点） | 全班 | 课中选择题、拍照上墙 | 参与率、选项分布、补录后正确率 |
| **课堂激活** | 活跃气氛、聚焦注意 | 少量定向 | 抢答、随机点名 | 响应覆盖面、轮次均匀度 |
| **探索操作** | 自主探究、协作 | 全班/个人 | 小程序/NB、互动课件/H5、智能体对话、小组讨论 | 成果提交率 |
| **教师教学** | 记录教师行为 | — | 发起互动、课件分享、讲评、投屏、AI对话、计时器、考勤 | 互动频次、互动时长 |


### 3.2 数据记录清单

> 每行 = 一条独立的数据记录。不规定字段命名，由开发设计表结构。
>
> **通用字段（所有记录默认包含，下表不再重复）：** 事件ID、课时ID、班级、学科、学生/教师ID、上报时间戳。
>
> **耗时字段：** 随堂测(#S2)、互动题(#S1)、课中选择题(#S4)、拍照上墙(#S5)、单词小测(#S3) 需额外记录"耗时"(int，秒，从互动发起到该生提交)，其他互动类型无需按学生记录耗时。

## 四、学情看板数据指标
> 线上逻辑：校级、班级、学生三级共用同一套卡片指标和互动日志结构，仅数据范围不同。

### 4.1 概览卡片（4个核心指标）

> **本期实际需求：** 按梳理出的11种学生互动，更新线上已有的"互动次数、参与率、正确率"指标。其中知识检测+开放检验计入**"题目互动"子类，其余互动计入"操作互动"子类。以下4指标为未来完整方向。

| 指标 | 数据来源 | 主数据计算规则 | 辅助数据 | 对比数据 |
|:-----|:---------|:--------------|:---------|:---------|
| 互动频次 | T1 | 筛选范围内 T1 count / 课时数 = X 次/课 | 累计互动次数：T1 count | 年级/班级平均 次/课 |
| 互动时长 | T1 | 筛选范围内所有 T1.互动耗时 求和 / 课时数 = X 分钟/课 | 累计互动时长：∑T1.互动耗时 | 年级/班级平均 分钟/课 |
| 课堂参与率 | S1~S11 | 除随机点名(T3)外的10类互动，每次单独算参与率再取均值。小组讨论以加入小组视为参与 | — | 年级/班级平均参与率 |
| 正确率 | S1~S4 | 筛选范围内所有知识检测+开放检验型互动的总正确题数 / 总作答题数 × 100% | — | 年级/班级平均正确率 |

**各层级数据范围：**
- 校级：筛选年级+学科+时段内所有班级
- 班级：当前班级+学科+时段
- 学生：当前学生个人数据，对比数据取班级平均

### 4.2 课堂互动日志 — 统计型（课时列表）

每课时一行，字段定义：

| 字段 | 数据来源 | 计算规则 |
|:-----|:---------|:---------|
| 日期 | 课时基础信息 | 课时日期 |
| 课节 + 开始时间 | 课时基础信息 | 如"第3节 (8:40)" |
| 互动次数 | T1 | 该课时内 T1 count |
| 参与率 | S1~S11 | 该课时内除随机点名(T3)外的10类互动，每次单独算参与率再取均值。小组讨论以加入小组视为参与 |
| 正确率 | S1~S4 | 该课时内所有知识检测+开放检验型互动的总正确题数 / 总作答题数 × 100% |

**学生视角差异：**
- "正确率"改为"个人正确率"：该生在 S1~S4 中的正确数 / 该生作答数

### 4.3 课堂互动日志 — 明细型（课时详情展开）

展开后按时间线逐条展示该课时每次互动，各类型辅助数据定义：

> **展示规则：** 所有互动统一用"参与人数/实到"形式展示参与情况，不显示计算后的百分比值。题数/单词数取教师发布的数量（全班一致），非学生端提交数累加。

| 互动类型 | 行为分类 | 辅助数据字段 | 数据来源 | 计算规则 |
|:---------|:---------|:------------|:---------|:---------|
| 随堂测 | 知识检测 | ① 题数 ② 参与人数/实到 ③ 正确率 | S2+T1+T11 | ① T1发布的题数（全班统一） ② 有S2记录学生数/T11签到数 ③ avg(已提交学生的S2.正确率) |
| 互动题(客观) | 知识检测 | ① 题数 ② 参与人数/实到 ③ 正确率 | S1+T1+T11 | ① T1发布的题数 ② 有S1记录学生数/T11签到数 ③ 已提交学生中S1.正确数/S1.总作答数 |
| 互动题(主观) | 知识检测 | ① 题数 ② 参与人数/实到 ③ 自批正确率 | S1.1+T1+T11 | ① T1发布的题数 ② 有S1.1记录学生数/T11签到数 ③ S1.1自批=对的数/已自批总数 |
| 单词小测 | 知识检测 | ① 单词数 ② 参与人数/实到 ③ 正确率 | S3+T1+T11 | ① T1发布的单词数（全班统一） ② 有S3记录学生数/T11签到数 ③ avg(已提交学生的S3.正确率) |
| 课中选择题 | 开放检验 | ① 参与人数/实到 ② 正确率 | S4+T4+T11 | ① 有S4记录学生数/T11签到数 ② 有T4时：S4.是否正确=true数/已提交人数 |
| 拍照上墙 | 开放检验 | ① 参与人数/实到 ② 上墙人数 | S5+T5+T11 | ① 有S5记录学生数/T11签到数 ② T5学生ID列表长度 |
| 抢答 | 课堂激活 | ① 参与人数/实到 ② 中签学生 ③ 幸运星 | S11+T2+T11 | ① S11中点击=true数/T11签到数 ② T2中签列表 ③ T2幸运星 |
| 随机点名 | 课堂激活 | ① 命中学生姓名 | T3 | T3.被点名学生ID → 查姓名 |
| NB实验 | 探索操作 | ① 参与人数/实到 ② 互动时长 | T1+T11 | ① 全班统一参与 ② T1.互动耗时 |
| 智能体对话 | 探索操作 | ① 参与人数/实到 ② 互动时长 | T1+T11 | 同NB实验 |
| 互动课件 | 探索操作 | ① 参与人数/实到 ② 互动时长 | T1+T11 | 同NB实验 |
| 小组讨论 | 探索操作 | ① 成果提交人数/实到 ② 上墙展讲人数 | S9+T11 | ① S9中提交=true学生数/T11签到数 ② S9中上墙=true学生数 |
| 小程序 | 探索操作 | ① 参与人数/实到 ② 互动时长 | T1+T11 | 同NB实验 |

### 4.4 校级学情

> 校级 = 全年级/全校维度的聚合数据，服务于校长/教务。

**概览卡片：** 同 4.1，数据范围为筛选年级+学科+时段内所有班级。

**教师互动频次对比：**

| 字段 | 数据来源 | 计算规则 |
|:-----|:---------|:---------|
| 教师姓名 | 课时基础信息 | — |
| 互动频次(次/课) | T1 | 该教师筛选范围内 T1 count / 课时数 |
| 分类堆叠 | T1.互动类型 | 按知识检测/开放检验/课堂激活/探索操作/教师教学 5类分色统计 |

**班级明细数据表：**

| 字段 | 数据来源 | 计算规则 |
|:-----|:---------|:---------|
| 班级名称 | 课时基础信息 | — |
| 正确率 | S1~S4 | 该班筛选范围内总正确题数 / 总作答题数 |
| 参与率 | S1~S11 | 该班各课时参与率取均值 |
| 互动频次(次/课) | T1 | 该班 T1 count / 课时数 |

支持按各列排序，点击班级进入班级学情。

### 4.5 学生学情

> 学生 = 单个学生的个人数据，服务于教师/班主任，对比数据取班级平均。

**顶部概要信息：** 学生姓名 + 班级 + 学科 + 正确率 + 参与率 + 爱学值(本月) + 综合排名

**概览卡片：** 同 4.1，数据范围为该生个人，对比数据取班级平均。

**课堂互动日志 — 统计型：**

| 字段 | 数据来源 | 计算规则 |
|:-----|:---------|:---------|
| 日期 | 课时基础信息 | 课时日期 |
| 课节 + 开始时间 | 课时基础信息 | 如"第3节 (8:40)" |
| 参与互动 | S1~S11 | 该生有记录的互动次数 / 该课时总互动次数 |
| 个人正确率 | S1~S4 | 该生在该课时内的正确题数 / 作答题数 |

**课时详情（展开后）：** 按行为分类（知识检测/开放检验/课堂激活/探索操作）合并展示，每类一行汇总（共N次 + 参与N/N次 + 类型关键指标），具体样式见 demo。

**参与率趋势图：** 按周聚合个人参与率 + 班级平均对比线（近4周/近1学期切换）

## 五、数据采集与同步

| 数据类型 | 同步方式 |
|:---------|:---------|
| 作答结果、积分变动、抢答/点名结果、教师行为 | 实时同步 |
| 操作时长/轨迹（探索型） | 课后批量同步 |

**离线容错：** 网络中断→本地缓存→恢复后同步→失败重试3次→仍失败告警。教师未结束课堂→最后事件+5分钟自动聚合。

**教师补录：** 课后回顾入口 → 补录正确答案 → 系统计算正确率。可选操作，时效7天，支持覆盖修改。

## 附录：边界Case

| 场景 | 处理方式 |
|:-----|:---------|
| 课中网络中断 | 本地缓存→恢复后同步 |
| 教师未结束课堂 | 最后事件+5分钟自动聚合 |
| 中途切换/更换设备 | 以课时ID/学生身份关联，不产生新记录 |
| 学期初无历史数据 | 趋势图从第2课时开始 |
| 教师补录选错 | 支持覆盖，以最后一次为准 |
| 超7天补录时效 | 入口置灰 |
| 某类行为本周期无数据 | 卡片展示"--" |
| 整个班无课堂互动数据 | 空态提示 |
| 预警名单为空 | 正向提示 |
| 教师对比中某教师无数据 | 显示为0，不隐藏 |
| 学生换班 | 学生页展示全部历史(标注班级)，班级页仅统计在班期间 |
| 教师查看非授课班级 | 权限拦截 |
| 聚合超时 | 60秒阈值，超时分批 |
